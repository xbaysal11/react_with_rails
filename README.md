# Requirements
   - 1.Ruby
   - 2.NodeJS
   - 3.Rails
   - 4.Bundle

# Installation

## 1. Clone or Download repository
[Click to Download](https://github.com/xbaysal11/reDDiv/archive/master.zip) 

## 2. Install packages
```
$ cd react_with_rails
$ npm install
$ bundle install
```

## 3. Start project
Steps:
   - 1.```$ rails s ```
   - 2.Go to [http://localhost:3000/](http://localhost:3000/)
